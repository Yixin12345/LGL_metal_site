# Codex Website Recreation Brief

## Goal

Create a professional mining / metals / industrial company website inspired by modern Canadian mining-service websites. The site should feel credible, clean, corporate, and practical. It should work as a company profile / business-card website for a metals or mining-related business.

Primary domain target: `www.llmetaltech.com`

Secondary domain: `www.lglmetal.com` can redirect to the primary domain.

Do not copy any exact text, images, logos, or proprietary layout from another website. Recreate the general style: large industrial hero image, strong corporate headline, simple navigation, service cards, project/capability sections, and a professional contact area.

---

## Design Direction

### Overall Style

* Professional Canadian industrial / mining company look.
* Clean, modern, corporate, trustworthy.
* Not too flashy; prioritize credibility and clarity.
* Use large high-quality industrial imagery: mining site, metal fabrication, steel, equipment, engineers, warehouse, logistics, or processing facilities.
* Use dark navy, charcoal, steel gray, white, and one accent color such as copper, gold, or industrial orange.

### Visual Keywords

* Mining services
* Metal technology
* Industrial solutions
* Engineering
* Manufacturing
* Procurement
* Reliable supply chain
* Safety and quality
* Project execution

### Suggested Color Palette

* Background dark: `#0F172A` or `#111827`
* Main text dark: `#1F2937`
* Light background: `#F8FAFC`
* Steel gray: `#64748B`
* Accent gold/copper: `#C99700` or `#B87333`
* White: `#FFFFFF`

### Typography

* Use a modern sans-serif font.
* Suggested: Inter, Satoshi, Manrope, or similar.
* Headings should be bold, spacious, and corporate.
* Body text should be readable and not too small.

## Technical Requirements

### Recommended Stack

Use one of the following:

Option A:

* Next.js
* React
* TypeScript
* Tailwind CSS
* Framer Motion for subtle animations

Option B:

* Vite
* React
* TypeScript
* Tailwind CSS

For free and stable deployment, prefer:

* Vercel
* Netlify
* Cloudflare Pages

### Components to Build

* `Header`
* `HeroSection`
* `TrustBar`
* `AboutSection`
* `ServicesSection`
* `CapabilitiesSection`
* `IndustriesSection`
* `GallerySection`
* `QualitySafetySection`
* `CtaBanner`
* `ContactSection`
* `Footer`

### UX Requirements

* Fully responsive design.
* Desktop, tablet, and mobile optimized.
* Smooth scrolling navigation.
* Contact form should be visually complete, even if backend submission is added later.
* Use semantic HTML.
* Optimize images.
* Good contrast and accessibility.
* Buttons should have hover/focus states.
* Navigation links should scroll to sections.

### Animation Requirements

Use subtle animations only:

* Fade-in sections on scroll.
* Slight card lift on hover.
* Smooth hero text entrance.
* No excessive motion.

### SEO Requirements

Add metadata:

* Title: `LL Metal Tech | Metal & Industrial Solutions`
* Description: `LL Metal Tech provides reliable metal supply, fabrication, and industrial support for mining, construction, and manufacturing clients.`
* Open Graph title and description.
* Use proper H1, H2, H3 hierarchy.

### Performance Requirements

* Avoid heavy animation libraries unless needed.
* Use compressed images.
* Lazy-load gallery images.
* Keep Lighthouse performance high.
